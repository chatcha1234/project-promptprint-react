# สิ่งที่ต้องทำต่อไป (Next Steps)

ตอนนี้เรามีระบบสมาชิก (Login/Register) ที่สมบูรณ์แล้ว แต่ระบบสินค้ายังไม่สมบูรณ์เนื่องจากเราเพิ่งลบข้อมูลจำลอง (Hardcode) ออกไป นี่คือรายการสิ่งที่ควรทำต่อครับ:

## 1. ระบบจัดการสินค้า (Product Management) - ⭐️ สำคัญมาก

- [ ] **สร้าง API เพิ่มสินค้า (Create Product)**: เพื่อให้เราเพิ่มสินค้าใหม่ลง Database ได้
- [ ] **สร้าง API แก้ไข/ลบสินค้า (Update/Delete)**: เผื่อลงข้อมูลผิดหรือเลิกขาย
- [ ] **หน้า Admin (Frontend)**: ฟอร์มสำหรับกรอกข้อมูลสินค้า (ชื่อ, ราคา, รูปภาพ, รายละเอียด)

## 2. ระบบตะกร้าสินค้า (Shopping Cart) - ⭐️ สำคัญ

- [ ] **หน้าสินค้าและรายละเอียด**: ปรับหน้าแสดงผลสินค้าให้ดึงข้อมูลจาก Database จริง (ซึ่งตอนนี้ว่างอยู่)
- [ ] **ปุ่ม Add to Cart**: กดแล้วเก็บข้อมูลลง State หรือ Database
- [ ] **หน้าตะกร้า (Cart Page)**: แสดงรายการที่เลือก, ปรับจำนวน, ลบออก, คำนวณราคารวม

## 3. หน้า Profile ผู้ใช้

- [ ] **แสดงข้อมูล**: ชื่อ, อีเมล, ประวัติการสั่งซื้อ (ในอนาคต)
- [ ] **แก้ไขข้อมูล**: เปลี่ยนรหัสผ่าน, แก้ไขชื่อ

---

**คำแนะนำ**: เริ่มที่ข้อ **1. สร้าง API เพิ่มสินค้า** ก่อนครับ เพราะถ้าไม่มีสินค้า เราก็ทำหน้าแรกร้านค้าไม่ได้
